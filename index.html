Start From Zero

Make Your Own Portfolio

A complete step-by-step guide for total beginners. No experience needed.

1

Create a GitHub Account

Your online code storage — think of it like Google Drive but for code.

1. Go to github.com and click Sign Up.

2. Enter your email, create a username and password.

3. Verify your account through the email they send you.

4. That's it — you now have a GitHub account! ✅

💡 What is GitHub? GitHub is a website where developers store their code online. Each project you save there is called a Repository (or "repo" for short). It's like a folder for your entire project that lives on the cloud.
2

Find a Sample Portfolio to Start From

Don't start from scratch — use an existing template as your base.

On GitHub, you can search for free portfolio templates others have shared. Here is a great beginner-friendly one you can use right now:

cobiwave / simplefolio

A clean, beautiful and responsive portfolio template for developers.

⭐ 12k+ stars · HTML / CSS / JS
↗ View on GitHub

You can also search for more templates by typing portfolio template in the GitHub search bar.

3

Fork the Repository (Copy It to Your Account)

This makes a personal copy of the template under your GitHub account.

1. Open the template repo link above.

2. Click the Fork button at the top right of the page.

3. Click Create Fork — GitHub copies the whole project to your account.

💡 What is a Fork? Forking means "I want my own copy of this project." Your forked copy is now at github.com/YOUR-USERNAME/simplefolio
4

Download It to Your Computer (Clone)

Cloning brings the code from GitHub down to your local machine.

You need Git installed first. Download it here:

↗ git-scm.com — Download Git (Free)

After installing Git, open the Terminal (on Windows: search for Git Bash or Command Prompt) and run this command:

$ git clone https://github.com/YOUR-USERNAME/simplefolio.git

Replace YOUR-USERNAME with your actual GitHub username.

💡 What does Clone do? It downloads a full copy of the project folder onto your computer so you can edit it.
5

Open the Folder in VS Code

This is where you'll edit everything — your name, projects, colors, and text.

1. Open VS Code.

2. Click File → Open Folder.

3. Find and select the simplefolio folder you just downloaded.

4. The project files will appear on the left sidebar panel.

5. Open index.html to start editing your name, bio, and project links!

💡 You can also open VS Code directly from the terminal by navigating into the folder and typing code .
6

Save Your Changes to GitHub (Push)

After editing, you sync your changes back to GitHub using these commands.

Open the terminal inside VS Code (Terminal → New Terminal) and run these 3 commands in order:

① Stage all your changes (tell Git what you edited):

$ git add .

② Commit with a message (describe what you changed):

$ git commit -m "Updated my name and projects"

③ Push to GitHub (upload your changes):

$ git push

📤 git push — Uploads your local changes to your GitHub repo online.

📥 git pull — Downloads the latest changes from GitHub to your computer (use this if you work on multiple devices or collaborate with others).

📝 git status — Shows you what files you've changed. Useful to check before pushing.

7

Deploy to the Internet — Vercel

This makes your portfolio live at a public URL — completely free.

1. Go to vercel.com and click Sign Up.

2. Choose Continue with GitHub — this links your accounts together.

3. Once logged in, click Add New → Project.

4. You'll see your GitHub repos listed — click Import next to simplefolio.

5. Leave all settings as default and click Deploy.

6. Wait about 30 seconds — Vercel will give you a live link like:

https://simplefolio-yourname.vercel.app
🚀 Auto-Deploy: From now on, every time you git push a change, Vercel will automatically rebuild and update your live site — no extra steps needed!
